```text
Collection (Interface)
├── List (Interface) - Có thứ tự, cho phép duplicate
│   ├── ArrayList
│   ├── LinkedList
│   └── Vector
│
├── Set (Interface) - Không thứ tự, không duplicate
│   ├── HashSet
│   ├── LinkedHashSet
│   └── TreeSet (SortedSet)
│
└── Queue (Interface) - FIFO
├── PriorityQueue
└── ArrayDeque

Map (Interface) - Key-Value pairs
├── HashMap
├── LinkedHashMap
├── TreeMap (SortedMap)
└── Hashtable
```
